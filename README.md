#  Подсчёт автобусов на остановке

Веб-приложение для автоматического обнаружения и подсчёта автобусов на изображениях остановок общественного транспорта с использованием технологий компьютерного зрения и нейронных сетей. Проект выполнен в рамках учебной практики по дисциплине «Системы искусственного интеллекта».

# Описание

Приложение реализует клиент-серверную архитектуру и предоставляет следующий функционал:
- Загрузка изображений через веб-интерфейс (с поддержкой drag-and-drop).
- Автоматическое детектирование автобусов с помощью предобученной модели **YOLOv8n**.
- Визуализация результатов с нанесением ограничивающих рамок на обнаруженные объекты.
- Сохранение истории всех обработок в реляционной базе данных **SQLite**.
- Экспорт накопленной статистики в формат **Microsoft Excel**.

# Стек технологий

- **Python 3.11** — основной язык программирования
- **Flask** — веб-фреймворк для серверной части
- **Ultralytics YOLOv8** — модель детектирования объектов
- **OpenCV** — обработка и декодирование изображений
- **SQLite** — хранение истории запросов
- **OpenPyXL** — формирование Excel-отчётов
- **HTML / CSS / JavaScript** — пользовательский интерфейс
